Label memory mapping:
---------------------

 0x0000 : main
 0x0008 : loop
 0x001c : shortloop
 0x003c : ColorNotZero
 0x0060 : RandomizePoint
 0x0090 : SetPixel
 0x0098 : RandomizeColor
 0x00b0 : RandomizeCoord
 0x0100 : WaitSecond
 0x0108 : WaitLoop
 0x011c : WaitSeconds
 0x012c : WaitSecondsLoop
 0x0130 : WaitSecFrameLoop
 0x0154 : PixelSprite
 0x0156 : Points
